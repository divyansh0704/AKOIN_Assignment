# Institute of Digital Risk (IDR) Homepage

## Overview
A responsive, single-page website designed for the Institute of Digital Risk. The site features a modern tech-style UI, mobile-friendly navigation, and smooth scroll animations.

## Technologies Used
- HTML5 (Semantic Structure)
- CSS3 (Flexbox, Grid, Animations, Responsive Design)
- Vanilla JavaScript (Mobile Menu, Scroll Interactions)

## How to Run
Simply open the `index.html` file in any modern web browser (Chrome, Firefox, Edge, Safari).

## File Structure
- `index.html`: Main HTML structure
- `style.css`: Styling and layout
- `script.js`: Interactivity (Menu, Animations)
- `logo-explanation.txt`: Explanation of the logo design choice
